Moneycontrol App - Home Page
This app is runnig on Tornado web server, using MySQL database in backend to store data and Python is being used for data processing/cleaning/analysis.
  • Once you click to the above link it will redirect to url /crawl/.
  • Crawler's log is being stored in output/output-log.log file.
  • It will create database schema and table if required.
  • It will set the rule to the table that no data can be inserted apart from a predefined company list.
  • Please refere COMPANY_LIST variable in config.py for predefined list of company.
  • First it will push the companies list along with url to the database. Then it will iterate over each company and hit the relavant url in Async mode.
  • Once gets the response from the company url it will create bucket for P/E Ratio and will update the relavant columns in table for the respective company.
  • Using BeautifulSoup python module to process html content.
  • Once you clicked on above link it will redirect to the url /main/ to show tabular representation of data as per requiremt.
  • Data is stored in MySQL database in backend.
  • Using sqlalchemy python module to connect with database.
  • Database related configuration like username/password/host/port/schema can get configured in config.py file.
  • Database queries are alseo stored in config.py module.
  • Using pandas to create DataFrame and perform data manipulation from database.
